# Interview Challenges

This repository contains the technical interview challenges used by [Wilcore Technologies](https://wilcore.io) to evaluate candidates across several engineering disciplines. Each challenge lives in its own directory and includes a detailed README with setup instructions and task descriptions.

---

## Challenges

| Role | Directory |
|------|-----------|
| Data Analyst / Data Scientist | [`data-analyst-challenge/`](./data-analyst-challenge/) |
| Database Administrator | [`dba-challenge/`](./dba-challenge/) |
| DevOps Engineer | [`devops/`](./devops/) |
| Backend Engineer (Java) | [`java/spring-boot-challenge/`](./java/spring-boot-challenge/) |
| Backend Engineer (Ruby) | [`ruby-on-rails/`](./ruby-on-rails/) |

---

## Getting Started

Clone the repository and navigate to the challenge directory you've been assigned:

```bash
git clone https://github.com/wilcore-tech/interview-challenges.git
cd interview-challenges
```

Each subdirectory has its own `README.md` with full prerequisites, setup steps, and task descriptions. Start there.
